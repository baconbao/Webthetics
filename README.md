## Predict only of Webthetics model

### Prepare env

- Ubuntu 18.04

```
$ conda create -n webthetics
$ conda env list
$ conda activate webthetics
$ conda install caffe
```

## Run predict

```
(webthetics) $ python baconbao-predict.py
```
